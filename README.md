# Jogo-da-forca
Projeto de um jogo clássico de forca desenvolvido durante meu período de estágio, com foco em aprimorar habilidades em desenvolvimento front-end. A aplicação foi construída inteiramente no lado do cliente, reforçando conceitos fundamentais de interface e interação com o usuário.

Tecnologias Utilizadas

HTML5
CSS3
JavaScript (Vanilla JS)

Objetivo

Projeto de um jogo clássico de forca desenvolvido durante meu período de estágio, com foco em aprimorar habilidades em desenvolvimento front-end. A aplicação foi construída inteiramente no lado do cliente, com foco em manipulação do DOM e interatividade.

Principais conceitos trabalhados:

Manipulação do DOM
Lógica de programação
Eventos em JavaScript
Estruturação de interface
